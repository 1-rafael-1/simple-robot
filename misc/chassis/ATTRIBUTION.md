## About Proto-Tank

Proto-Tank is a tracked chassis design that serves as the foundation for the simple-robot's mobility platform. The design provides a robust and modular base for adding sensors, motors, and control electronics.

## Modifications

The Files of the original Proto-Tank design have been included in this repository unmodified for ease of access. Based on the original design, modified and additional files have been created to better integrate with the simple-robot project. These modifications include:

+ Added a modified base frame, with filled out bottom and cutouts to accomodate the motor encoders.
+ Added a 9g servo mount
+ Added a PCB mount for the simple-robot main board

The original files can be found in the `misc/chassis/proto-tank/` directory, while the modified files are located in the `misc/chassis/` directory.
